Project Overview

Webserv is an HTTP server developed to provide a reliable and efficient service. It was created as a learning project to understand and implement core HTTP concepts.

Project History

Webserv started as a project to understand and implement the core concepts of HTTP. Over time, it evolved into a fully-functional HTTP server with a focus on performance and reliability.
