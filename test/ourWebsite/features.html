Core Features

  • Support for GET, POST, DELETE methods
  • File uploads
  • Multiple port listening
  • Non-blocking I/O operations
  • Poll() system call for handling connections

Technical Details

Webserv uses non-blocking I/O operations and the poll() system call to handle multiple connections efficiently. It is written in C++ 98 and ensures compatibility with major web browsers.

Example configuration:


server {
    listen 80;
    server_name example.com;
    root /var/www/html;
    ...
}
