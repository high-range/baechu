Installation Guide

To install Webserv, follow these steps:


git clone https://github.com/your-repo/webserv.git
cd webserv
make
./webserv config.conf

Usage Example

Here is an example configuration file:


server {
    listen 80;
    server_name example.com;
    root /var/www/html;
    ...
}
